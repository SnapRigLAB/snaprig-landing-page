## üÜò TROUBLESHOOTING

### "BUY NOW still redirects to homepage"
**Solution:** 
- Make sure password protection is OFF
- Make sure Shopify Payments is ACTIVE (not just "in review")
- Try: `snapr-3.myshopify.com/checkout` - does it work?
- If not, Shopify Payments isn't fully active yet

### "Checkout says 'Store not set up'"
**Solution:**
- Complete ALL legal policies
- Activate Shopify Payments
- Remove password protection

### "Customers can't pay"
**Solution:**
- Check Settings ‚Üí Payments
- Make sure Shopify Payments status is "Active" (green)
- If "Under review" - wait for approval (1-3 days)

### "Order doesn't appear in Shopify"
**Solution:**
- Check if checkout test mode is on
- Settings ‚Üí Payments ‚Üí Disable test mode
